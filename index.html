DEFINICIÓN

Una variable aleatoria es una función que asigna a cada valor del espacio muestral un valor numérico real. La V.A. asigna un número a cada suceso elemental del espacio muestral, es decir, cuantifica una característica. Dichos valores forman el recorrido de la variable. Normalmente se usan letras mayúsculas, y del final de abecedario, (X, Y, o Z) para denotar variables aleatorias. De un experimento aleatorio es posible definir más de una variable aleatoria.

PROPOSICIÓN

Para 2 números cualesquiera a y b con a ≤ b, se cumple que:

P(a ≤ x ≤ b) = F(b) - F(a)

Donde “a-“ representa el valor posible de X más grande que es estrictamente menor que a. En particular, si los únicos valores posibles son enteros y si a y b son enteros, entonces

P(a ≤ x ≤ b) = P(X = a 0 a +1 o a+2 o ... o b) = F(b) - F(a - 1)

Con a =b se obtiene P(X = a) =F(a) – F(a – 1)
